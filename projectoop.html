Usage of libraries

| pandas: Used for data manipulation and analysis. Ideal for handling tabular data with DataFrames, allowing filtering, grouping, merging, and more.
| numpy: Fundamental package for numerical computing in Python. Provides support for arrays, mathematical functions, linear algebra, and random number generation.
| matplotlib.pyplot (plt): Core plotting library for Python. Helps create basic graphs such as line plots, bar charts, scatter plots, etc.
| seaborn (sns): High-level interface for drawing attractive statistical graphics. Built on top of Matplotlib, it simplifies complex visualizations.
| unittest.mock (Base): Part of Pythons standard testing library. Allows you to mock objects and functions during testing.
| train_test_split from sklearn.model_selection Used to split the dataset into training and testing parts. Helps evaluate how well a model generalizes to unseen data.
| SVC from sklearn.svm: Implements Support Vector Classification, a powerful machine learning algorithm for classification tasks.
| sklearn.metrics:
  • Evaluation metrics:
  • classification_report gives precision, recall, F1-score.
  • confusion_matrix shows actual vs predicted results.
  • accuracy_score measures overall prediction accuracy.
| pickle: Used to serialize (save) and deserialize (load) Python objects, such as trained machine learning models. Useful for storing and reusing models without retraining.

class DataLoader:

            print(f"Data loaded successfully with shape: {data.shape}")
            return data

import numpy as np

class SimpleSVM:
    def __init__(self, C=1.0, learning_rate=0.001, n_iters=1000):
        self.C = C
        self.lr = learning_rate
        self.n_iters = n_iters
        self.w = None
        self.b = None

    def split_data(self, X, y, test_size=0.2, random_state=42):
        """
        Simple train-test split without sklearn.
        """
        np.random.seed(random_state)
        indices = np.arange(X.shape[0])
        np.random.shuffle(indices)
        split = int(X.shape[0] * (1 - test_size))
        train_idx, test_idx = indices[:split], indices[split:]
        return X[train_idx], X[test_idx], y[train_idx], y[test_idx]

    def create_model(self, n_features):
        """Initialize weights and bias."""
        self.w = np.zeros(n_features)
        self.b = 0
        print("Model created with weights initialized to zero.")

    def train(self, X, y):
        """Train the model using gradient descent for hinge loss."""
        for _ in range(self.n_iters):
            for idx, x_i in enumerate(X):
                condition = y[idx] * (np.dot(x_i, self.w) - self.b) >= 1
                if condition:
                    self.w -= self.lr * (2 * self.w)
                else:
                    self.w -= self.lr * (2 * self.w - np.dot(x_i, y[idx]) * self.C)
                    self.b -= self.lr * y[idx]
        print("Training completed.")

    def test(self, X, y):
        """Evaluate model accuracy."""
        y_pred = np.sign(np.dot(X, self.w) - self.b)
        accuracy = np.mean(y_pred == y)
        print(f"Test Accuracy: {accuracy*100:.2f}%")

# Step 2: Create and train the model

X_train, X_test, y_train, y_test = svm.split_data(X, y)

svm.train(X_train, y_train)
svm.test(X_test, y_test)

VISUALIZTION OF BIVARIANT

Heatmap

The purpose of a heatmap in data science especially a correlation heatmap is to visually represent the relationship between variables in a dataset.

Heatmap (specifically correlation heatmap):

Purpose:

  • To show how strongly each pair of features (columns) are related to each other.

  • To help identify patterns, trends, or potential multicollinearity in your data.

Why it's useful:

  • Spot redundant features (e.g., if two features are >0.9 correlated, maybe drop one).

  • Select important features for models.

  • Understand the structure of your data before training models.

Scatter plot

A scatter plot shows the relationship between two numeric features, helping visualize trends, patterns, and potential outliers.

Purpose of a Scatter Plot:

  • To visualize relationships, trends, or patterns between two variables.

  • To spot correlations (positive, negative, or none).

To detect:

  • Outliers

  • Clusters

  • Linear/non-linear trends

def plot_correlation_heatmap(self):
        corr = self.df.corr(numeric_only=True)
        cmap = sns.diverging_palette(220, 20, as_cmap=True)
        plt.figure(figsize=(12, 5))  # shorter height
        sns.heatmap(
            corr, 
            annot=True, 
            fmt='.2f', 
            cmap=cmap, 
            linewidths=0.5, 
            linecolor='white', 
            square=True, 
            cbar_kws={'shrink': 0.8}
        )
        plt.title('Correlation Heatmap', fontsize=16)
        plt.xticks(rotation=45)
        plt.yticks(rotation=0)
        plt.tight_layout()
        plt.show()

Histogram

A histogram is a type of graph that shows the distribution of a single numerical variable by dividing the data into intervals (called bins) and counting how many values fall into each bin.

Purpose of a Histogram:

  • To visualize how data is spread (i.e., distribution).

  • To detect:

  • Skewness (left, right, or symmetric)

  • Outliers

  • Peaks (modes) where data is concentrated

  • Helps you understand the shape of the data: normal, skewed, bimodal, etc.

import pandas as pd
import matplotlib.pyplot as plt

class DataVisualizer:
    def __init__(self, df):
        self.df = df

    def plot_histograms(self, numeric_cols):
        plt.figure(figsize=(14, 5))
        self.df[numeric_cols].hist(
            bins=30,
            figsize=(14, 5),
            color='skyblue',
            edgecolor='black'
        )
        plt.suptitle('Distribution of Numeric Features', fontsize=16)
        plt.tight_layout()
        plt.show()
